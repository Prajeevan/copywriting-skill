# Copywriting Skill

A reusable [Agent Skill](https://skills.sh) that teaches AI agents to write persuasive, high-converting copy — ads, headlines, landing pages, email, VSL/short-form video scripts, product descriptions, and CTAs.

Built on classic direct-response frameworks (Ogilvy, Hopkins, Sugarman, the Wall Street Journal "Two Young Men" ad, Cialdini) as distilled by **Sam Parr** (founder of *The Hustle*, co-host of *My First Million*).

> **Credit:** The frameworks in this skill come entirely from **Sam Parr** in the episode **"Become a Dangerously Good Writer in 55 Minutes w/ Sam Parr"** on the **[Sweat Equity](https://www.youtube.com/@SweatEquityPodcast)** podcast. Watch it here: **https://www.youtube.com/watch?v=uf4fR3qcDkU**. This repo is a study aid; all original insight is theirs.

## What's inside

- **Core philosophy** — "no such thing as too long, only too boring"; write for the *right* reader, not everyone; niches make riches
- **Headlines & hooks** — the curiosity gap, open loops, punch-in-the-face first lines, targeting, twisting known phrases, making the abstract concrete
- **Slippery slope body copy** — micro-commitments, the Trojan-horse product reveal
- **Storytelling** — customer = hero / brand = guide, "but/therefore" not "and-then", failed-solution guilt absolution, framebreaking
- **A worked AIDA ad**, objection handling (the 8 Mile move, anecdotes, FAQ), proof & social proof
- **Mechanics** — rhythm, 7th-grade reading level, Buffett's 17-words-per-sentence
- **Copy Work** — the by-hand imitation practice + swipe files
- A pre-ship checklist

## Install

```bash
npx skills add Prajeevan/copywriting-skill
```

Then ask your agent to write or sharpen any piece of marketing/sales copy and it will apply these frameworks.
